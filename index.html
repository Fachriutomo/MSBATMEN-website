Project Saya

Untuk membuat sebuah website atau laman pribadi itu penuh tantangan, karena semua akan diuji dengan hal yang namanya BUG atau ERROR.
Saya pribadi sudah mencoba kedua kalinya, pertama saya membuat sebuah website yang sangat simpel namun itu masih belum cukup karena saya masih melihat tutorial dari youtube, tetapi lama-kelamaan saya mulai mendalami lagi apa yang sebenarnya dipelajari.

dalam memyusun atau membuat sebuah laman atau web pasti akan menggunakan bahasa pemrograman sebagai berikut

HTML adalah bahasa pemrograman yang dipahami setiap pengembang web. HTML merupakan tulang punggung atau struktur dasar dalam halaman web, mengatur elemen-elemen, dan menghubungkan satu sama lain. HTML bekerja dengan menggunakan tag (penanda) yang diletakkan di dalam tanda '<' dan '>'.

contoh Struktur dasar HTML
!DOCTYPE html
html
head
title Judul Halaman /title
/head
body
h1 Selamat Datang di Situs Web Saya /h1
p Ini adalah contoh paragraf. /p
body
/html

sumber: www.kmtech.id/post/

CSS adalah bahasa yang digunakan untuk mengatur tampilan atau gaya suatu halaman web yang telah dibangun menggunakan HTML. Dengan CSS, pengembang dapat mengontrol warna, ukuran, tata letak, dan elemen visual lainnya pada halaman. Jika HTML bertanggung jawab atas struktur, maka CSS atau Cascading Style Sheets bertanggung jawab atas tampilan visual dari halaman web.

Untuk memahami CSS maka Aturan gaya harus terdiri dari pasangan "nama properti" dan "nilai properti". Contoh sederhana:
p {
color: blue;
font-size: 16px;
margin-bottom: 20px;
}

sumber: www.kmtech.id/post/

JavaScript adalah bahasa pemrograman yang digunakan untuk memberikan interaktivitas pada halaman web. Dengan JavaScript, Anda dapat mengontrol perilaku halaman, mengambil dan memanipulasi data, serta merespons interaksi pengguna. JavaScript memberikan kemampuan kepada pengembang web untuk membuat efek animasi, validasi formulir, dan merespons tindakan pengguna secara real-time.

sumber: www.kmtech.id/post/
